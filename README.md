🧠 Machine Learning Overview

Machine Learning enables systems to learn patterns from data and make predictions or decisions without being explicitly programmed.

Types:
Supervised Learning
Unsupervised Learning
Reinforcement Learning
📈 Linear Regression
Simple Linear Regression
Predicts a target variable using a single feature
Establishes a linear relationship between input and output
📊 Multiple Linear Regression
Extends linear regression to multiple input features
Improves prediction accuracy with more variables
📏 Model Evaluation using R² Score
Measures how well the model explains variance in the data
Values range from 0 to 1 (higher is better fit)
🔤 One Hot Encoding

Converts categorical variables into numerical format.

Methods:
Using pandas (get_dummies)
Using sklearn (OneHotEncoder)
🔀 Train-Test Split
Splits dataset into training and testing subsets
Ensures proper evaluation of model performance
🎯 Logistic Regression
Used for classification problems
Outputs probabilities mapped to class labels
📉 Confusion Matrix

Evaluates classification performance using:

True Positive
False Positive
True Negative
False Negative
🌳 Decision Tree
Tree-based model used for classification and regression
Easy to interpret and visualize
📐 Support Vector Machine (SVM)
Finds the optimal hyperplane to separate classes
Effective in high-dimensional spaces
🌲 Random Forest
Ensemble learning method using multiple decision trees
Reduces overfitting and improves accuracy
🔁 Cross Validation
Improves model reliability by validating across multiple data splits
K-Fold Cross-Validation
Divides data into k subsets and trains/testing iteratively
🔍 K-Means Clustering
Unsupervised learning algorithm for grouping data
Key Steps:
Determine optimal number of clusters using the elbow method
Use features like petal length and petal width (Iris dataset)
Remove irrelevant features for simplicity
